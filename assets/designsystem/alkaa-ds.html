Alkaa uses a single consistent icon style throughout — outlined stroke icons at 2dp stroke width with round line caps and joins. Icons are never filled (except interactive star/favourite states). All sizes below are in density-independent pixels (dp).

Dos & Don'ts
Do
Use outlined icons consistently
Pair icons with a text label where space allows
Use filled variant only for active toggle states
Maintain 48×48dp minimum touch target
Don't
Mix outlined and filled icons arbitrarily
Scale icons below 16dp or above 32dp
Use icons without sufficient colour contrast
Rasterize SVG icons or use <img> for UI icons
