# Libraries

Layer responsible for providing common code, utilities, and design system components used across the entire application.

## Modules

* **designsystem**: The "Source of Truth" for the app's look and feel, containing themes and reusable UI components.
* **navigation**: Common navigation logic and structures.
* **di**: Dependency injection configuration.
* **test**: Utilities and base classes for testing.
* **coroutines**: Dispatchers and coroutine-related utilities.
* **permission**: Wrapper for handling system permissions.
